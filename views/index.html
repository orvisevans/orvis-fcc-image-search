Image Search with search history

Part 4 of Free Code Camp API Challenges

Search by appending '/api/imagesearch/' then a search term to the page's url to access the backend image search api. Paginate the results by appending the querystring "?offset=10" or however many items to offset by. Like: https://orvis-fcc-image-search.glitch.me/api/imagesearch/doggos?offset=1

Also see recent searches by visiting https://orvis-fcc-image-search.glitch.me/api/latest/imagesearch/
